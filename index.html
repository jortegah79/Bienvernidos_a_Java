Primeros pasos con Java

JAVA fué creado en 1991 por SUN MICROSYSTEMS,aunque su primera versión salió a la luz en 1996. Es un lenguaje de propósito general y orientado a objeto.

Puede ser utilizado en diversos sistemas operativos, puesto que podemos implementar aplicaciones en Windows,Linux y Mac.

Otra característica es que el colector de basura es automático. A difeferencia de c++,por ejemplo, no tenemos que preocuparnos de liberar de la memória las variables declaradas.

Java no es JAVASCRIPT. Puede llevar a confusión y a pesar de que su sintaxis es parecida, a medida que avanzamos vemos mayores diferencias entre ellos.

Estructura básica

Todo programa en java necesita un método main o un método init(si hablamos de java en un servidor).Todo ello englobado en una clase principal.


                    public class ClasePrincipal{
                        public static void main(String [] args){
                            System.out.println("Hola mundo");
                        }
                    }

Este sería nuestro primer programa con Java.Un "hola mundo".Como vemos, debe de ir entre llaves.La clase engloba todo y cada método engloba el grupo de instrucciones que le pertenecen.

TIpo de dato primitivo

Hay 8 tipos de dato primitivo. String,char,float,double,short,long,boolean y byte.

Operadores

Podemos utilizar los diferentes operadores matematicos habituales. La suma, resta, multiplicación y división. Agregamos el resto(%) que nos dá el resto de una división.int x= 10 %3; El resultado será 1.

También tenemos los operadores lógicos or y and y la negación, indicada con (!) delante de la sentencia.

Condicionales

Para ofrecer opciones al código, podemos utilizar los condicionales.Estos pueden ser los if, los if-else o los switch-case. if(x==5){
System.out.println("X es igual a 5"){
}else{
System.out.println("no es igual a 5")
}
.EL if-else es un condicional y si se cumple, se ejecuta una acción, pero si no se cumple, se ejecuta otra acción. El else es opcional. También podemos encadenar varios if.
