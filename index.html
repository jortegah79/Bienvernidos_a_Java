Primeros pasos con Java

JAVA fué creado en 1991 por SUN MICROSYSTEMS,aunque su primera versión salió a la luz en 1996. Es un lenguaje de propósito general y orientado a objeto.

Puede ser utilizado en diversos sistemas operativos, puesto que podemos implementar aplicaciones en Windows,Linux y Mac.

Otra característica es que el colector de basura es automático. A difeferencia de c++,por ejemplo, no tenemos que preocuparnos de liberar de la memória las variables declaradas.

Java no es JAVASCRIPT. Puede llevar a confusión y a pesar de que su sintaxis es parecida, a medida que avanzamos vemos mayores diferencias entre ellos.

Estructura basica

Todo programa en java necesita un método main o un método init(si hablamos de java en un servidor).Todo ello englobado en una clase principal. 


public class ClasePrincipal{
    public static void main(String [] args){
        System.out.println("Hola mundo");
    }
}

Este sería nuestro primer programa con Java.Un "hola mundo".Como vemos, debe de ir entre llaves.La clase engloba todo y cada método engloba el grupo de instrucciones que le pertenecen.

Tipos de dato

Java es un lenguaje con tipado estático y fuerte. Esto significa que una vez declaremos una variable con un tipo, este no cambiará.Estas cosas lo convierten en un lenguaje robusto y tolerante a fallos.

Los tipos primitivos de datos son:

  • Números enteros

  • byte
  • short
  • int
  • long

  • Números reales

  • float
  • double

  • char o character

  • boolean.

Además, hay un dato compuesto que se usa tanto que casi podriamos considerarlo un primitivo(En otros lenguajes lo és): EL STRING O CADENA

La cadena es un grupo de chararacter formando una cadena.


char letra='a';
String palabra="aaaaa";

El char se escribe entre comilla simple y el string entre dobles comillas.

También tenemos los operadores <(menor que) ,> (mayor que), <=( menor o igual que) o=>(mayor o igual que) 


int suma=4 +3;          //resultado :7;

int division=4/3;       //resultado :1;

Constantes y variables

Un punto importante es conocer las variables y las constantes. Una variable es una porción de memoria que declaramos y que durante la ejecución del programa, puede tener diferentes valores. Una constante tiene siempre un mismo valor.

Para darles un valor, utilizamos el signo de asignación (=)


int x=5;            //Esto es una variable.
final int v=10;     //Esto es una constante.
x=10;               //correcto;
v=14;               //error.

Condicionales

Para ofrecer opciones al código, podemos utilizar los condicionales.Estos pueden ser los if, los if-else o los switch-case.

Los condicionales son como opciones para el código. Si se cumple una condición, realizaremos una serie de instrucciones. 

                   
int x=5;
if(x > 3){
    Sytem.out.println("Es mayor que 3");
}

En este código podemos ver que si x es mayor que 3, se cumple la condición, por lo tanto en pantalla nos dira "Es mayor que 3".Pero si no se cumple, no tendremos ningún mensaje.Si queremos que nos salga, podriamos utilizar un if-else. 


int x=10;
if(x < 5){
    System.out.println("El número es mayor");
}else{
    System.out.println("El número es menor");
}

Aquí hemos visto que podemos controlar 2 estados.Podriamos encadenar diferentes condiciones, sin ningún problema, aunque es posible que con un switch-case, la sensación del código seria mas elegante. 


int x=2;
switch(x){
    case 0:
        System.out.println("El número es 0");
        break;
    case 1:
        System.out.println("El número es 1");
        break;
    case 2:
        System.out.println("El número es 2");
        break;
    default:
        System.out.println("El número no es 0 ni 1 ni 2");
        break;
}

Nos muestrará que el número es el 2.

Estructuras repetitivas

Para realizar tareas repetitivas o en bucle, tenemos una serie de estructuras muy utiles:

  • WHILE
  • DO-WHILE
  • FOR
  • FOR-EACH

Hay 2 tipos de estructura. Las que sabemos cuantas veces se van a repetir y las que no. Entre las que no tenemos un número de repeticiones definido tenemos los 2 bucles while.

WHILE es un bucle que realiza su función MIENTRAS se cumple una sentencia. 


int x=0;
while( x < 2){
    System.out.println(x);
    x++;
}

Hará repeticiones mostrandonos el número que contiene la x. Si cuando llegue a la condición, se sigue cumpliendo, vuelve a ejecutarse el código.Esto nos mostrará 0,1 y el 2 no, porque no se cumpliria la condición de ser menor que 2.

DO-WHILE funciona igual, pero siempre ejecutará el código 1 vez mínimo, porque hace la comprobación después del bloque de código. 


boolean datoOk=false;
do{
    datoOk=true;

}while(datoOk==false);

Este es un ejemplo absurdo, pero rápido. Entramos con una variable que tiene un false como valor. Nuestra condición es que esa variable tenga un valor false, pero en el interior del código cambiamos el valor a true.Hemos ejecutado las sentencias del bloque 1 sola vez, pero se ha ejecutado.

Entre las que conocemos la cantidad de repeticiones que se realizarán, la principal es el bucle for, en el que seremos nosotros quienes determinemos cuantas veces ejecutaremos el codigo interno. 


for(int i=0;i < 5; i++){
    System.out.print(i+ ",")
}
System.out.print(5);

Este código nos imprimirá los números del 0 al 4 con una coma intermedia ( 0,1,2,3,4,) y nos agregará el 5 con el print que está fuera del bucle, porque tenemos controlado el resultado.

Dejaremos para otros proyectos el bucle for, destinado sobretodo para el uso sobre arrays u objetos.
